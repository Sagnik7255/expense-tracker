# Expense Tracker
A simple expense tracker that tracks incoming and outgoing transaction details (like name, amount, etc) to track expenses.

## Project Overview

**Tech Stack:**
- Frontend: React (Vite)
- Backend: Node.js + Express
- Storage: In-memory (No database)

## Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/devsoc-aec/expense-tracker.git
```

### 2. Install dependencies
**Server**
```bash
cd server
npm install
npm run dev
```

**Client**
```bash
cd client
npm install
npm run dev
```
